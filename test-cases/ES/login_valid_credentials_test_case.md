# Caso de Prueba

## ID del Caso de Prueba
TC-01

## Titulo
Verificación de inicio de sesión con credenciales válidas 

## Precondiciones
El usuario se encuentre registrado en el sistema.

## Pasos
1. Abrir la aplicación
2. Ingresar un correo electrónico válido
3. Ingresar una contraseña valida
4. Presionar el bóton de "Login"

## Resultado esperado
El sistema permite el acceso al usuario y lo redirige a la página principal.

## Resultado real
(Por completar durante la ejecución)

## Estatus
Aprobado / Fallido
