# Bug Report

## Title
Email field accepts invalid email format

## Description
The email input field allows invalid email formats without displaying a validation message.

## Environment
- Application: Web system
- Browser: Google Chrome
- Operating System: Windows 10

## Steps to Reproduce
1. Open the application
2. Enter "test@@" in the email field
3. Enter any password
4. Click the "Submit" button

## Expected Result
The system should display a validation message indicating an invalid email format.

## Actual Result
The system allows submission without showing any validation error.

## Severity
Medium

## Priority
Medium

## Evidence
Not available (manual testing)
