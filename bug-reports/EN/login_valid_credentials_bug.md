# Bug Report

## Note
This bug report is included as an example of critical login issue documentation.
The behavior described does not occur in the SauceDemo application.

## Title
Login fails when using valid credentials

## Description
The system does not allow the user to log in even when valid credentials are entered.

## Environment
- Application: Web system
- Browser: Google Chrome
- Operating System: Windows 10

## Steps to Reproduce
1. Open the application
2. Enter a valid email address
3. Enter a valid password
4. Click on the "Login" button

## Expected Result
The system should allow access and redirect the user to the main page.

## Actual Result
The system displays an error message and does not allow access.

## Severity
High

## Priority
High
