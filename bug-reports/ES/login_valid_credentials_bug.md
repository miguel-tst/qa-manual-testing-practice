# Reporte de Bug

## Título
El inicio de sesión falla al usar credenciales válidas

## Descripción
El sistema no permite el inicio de sesión aun cuando se ingresan credenciales válidas.

## Entorno
- Aplicación: Sistema web
- Navegador: Google Chrome
- Sistema Operativo: Windows 10

## Pasos para Reproducir
1. Abrir la aplicación
2. Ingresar un correo electrónico válido
3. Ingresar una contraseña válida
4. Presionar el botón "Iniciar sesión"

## Resultado Esperado
El sistema permite el acceso y muestra la pantalla principal.

## Resultado Real
El sistema muestra un mensaje de error y no permite el acceso.

## Severidad
Alta

## Prioridad
Alta

## Evidencia
No disponible (prueba manual)
